# Candidate Manual

## Candidate Journey

1. Receive invitation email.
2. Open secure assessment link before expiry.
3. Verify OTP if required.
4. Read instructions.
5. Complete compatibility check.
6. Start assessment.
7. Use timer, navigator, flag, and review page.
8. Submit manually or wait for auto-submit on timeout.
9. Receive confirmation.
10. Download certificate if enabled and eligible.

## Candidate Protections

- Answers are autosaved.
- Candidate can resume after disconnect if the assessment window remains open.
- Browser refresh is protected.
- Timer and grace period are visible.
- Accessibility and keyboard navigation are supported.

## Candidate Restrictions

- Each secure link is one-time use.
- Tab switching, copy/paste, fullscreen exit, right click, and suspicious device events may be recorded.
- Sharing a link or attempting from an unauthorized device may block the attempt.
