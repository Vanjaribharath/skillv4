# Admin Manual

## Admin Responsibilities

- Configure organization profile and branding.
- Manage departments and batches.
- Create and manage trainers and candidates.
- Configure SMTP and email templates.
- Manage roles and permissions.
- Monitor audit logs, health dashboard, imports, exports, and jobs.
- Configure integrations and webhooks.
- Review backups and restore readiness.

## Common Workflows

### Organization Setup

1. Register organization.
2. Upload logo and choose brand colors.
3. Configure certificate template.
4. Configure SMTP.
5. Create departments and batches.
6. Add trainers and candidates.

### Health Monitoring

Admins review:

- API health.
- Database connectivity.
- Redis connectivity.
- RabbitMQ queue depth.
- Email delivery failures.
- Export and certificate job status.
- Backup status.

### Audit Review

Admins can filter audit logs by actor, action, entity, date range, IP address, and organization.
