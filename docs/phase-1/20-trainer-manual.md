# Trainer Manual

## Trainer Responsibilities

- Create and maintain questions.
- Submit questions for review.
- Build assessment templates and assessments.
- Schedule assessments.
- Invite candidates.
- Monitor live attempts.
- Review suspicious activity.
- Manually evaluate coding or subjective answers.
- Generate reports and certificates.

## Assessment Creation Workflow

1. Choose an assessment template or create a new assessment.
2. Add sections such as Linux, Shell, and Java.
3. Select question pools, difficulty distribution, marks, and negative marking.
4. Set duration, access window, grace period, and candidate limit.
5. Preview the assessment.
6. Publish.
7. Invite candidates by batch, individual selection, or CSV.

## Live Dashboard

The live dashboard shows candidates as not started, started, in progress, disconnected, completed, or flagged. Trainers receive submission, disconnection, and suspicious activity alerts.
