# Release Notes

## Version 1.0.0 Planned Scope

SkillForge Enterprise v1.0.0 will include the first production release of the internal enterprise assessment platform.

### Planned Features

- Multi-organization support.
- Admin, trainer, evaluator, candidate roles.
- Organization branding.
- Question bank with versioning, approval, import, export, and analytics.
- Assessment templates, sections, randomization, scheduling, invitations.
- Secure candidate magic links and OTP verification.
- Autosave, resume, auto-submit, grace period.
- Anti-cheating event tracking.
- Automatic and manual scoring.
- Reports, exports, leaderboards, certificates.
- Email templates, reminders, in-app notifications.
- REST APIs, webhooks, audit logs.
- Health dashboard, backup tracking, Docker deployment.

### Known V1 Limits

- Webcam and microphone support are optional and may be delivered as controlled add-ons.
- AI question generation and suspicious scoring require review controls before production use.
- Native mobile apps are not included.
