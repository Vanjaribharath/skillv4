# Version Roadmap

## V1: Enterprise Internal Assessment Core

- Full assessment lifecycle.
- Trainer and admin workspaces.
- Secure candidate player.
- Reports, exports, certificates.
- Deployment and operations baseline.

## V1.1: Content And Workflow Expansion

- Larger seed question packs.
- More import validators.
- Saved filters and trainer dashboard personalization.
- Localization for high-priority languages.
- Advanced certificate designer.

## V1.2: Integrations

- LMS integrations.
- HRMS integrations.
- SCIM provisioning.
- SSO enhancements.
- Webhook marketplace examples.

## V2: Advanced Proctoring And AI

- Optional webcam and microphone proctoring.
- AI-assisted question generation with review workflow.
- AI-assisted weak-area coaching summaries.
- Plagiarism checks for coding answers.

## V3: SaaS Commercialization

- Billing.
- Plan limits.
- Customer admin console.
- Multi-region deployment.
- Marketplace content packs.
