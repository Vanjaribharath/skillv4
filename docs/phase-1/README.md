## Brand Summary

SkillForge Enterprise is an internal assessment and certification platform for companies, universities, corporate trainers, bootcamps, and learning teams. The product emphasizes reusable assessments, governed question banks, organization branding, auditability, analytics, and secure candidate delivery.

Logo concept: a shield-shaped forge mark containing a checkmark and bracket symbol, representing trusted assessment plus technical skill validation.

Color palette:

| Token | Color | Usage |
| --- | --- | --- |
| Primary | `#1D4ED8` | Navigation, primary actions |
| Secondary | `#0F766E` | Success states, skill growth |
| Accent | `#F59E0B` | Warnings, scheduled events |
| Surface | `#F8FAFC` | Page background |
| Ink | `#111827` | Primary text |
| Muted | `#64748B` | Secondary text |
| Danger | `#DC2626` | Risk and security alerts |

Typography guide:

| Role | Font | Weight |

| Reports/PDF | Inter or Noto Sans fallback | 400, 600 |

## Design System Principles

- Dense enterprise layouts for trainers and admins; calm, guided experience for candidates.
- Cards only for repeated entities such as assessments, questions, candidates, reports, and audit entries.
- Icons for common actions: publish, clone, preview, export, invite, archive, approve, restore.
- Accessible contrast, keyboard navigation, visible focus, screen-reader labels, and responsive layouts.
- Organization-level theming for logo, primary color, certificate template, and email templates.
